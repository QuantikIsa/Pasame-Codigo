## Licencia correctamente tus contribuciones

A la hora de contribuir con código a este repositorio, por favor añade en la cabecera de los archivos que subes el siguiente

```c title=""
// SPDX-FileCopyrightText: 2024 <nombre> <apellido1> <apellido2> <tu.correo@mail.com>
//
// SPDX-License-Identifier: GPL-3.0-only
```

El código contribuido directamente a este repositorio es `GPL-3.0-only`, de ser importado de otro repositorio adecúa correctamente la cabecera.
